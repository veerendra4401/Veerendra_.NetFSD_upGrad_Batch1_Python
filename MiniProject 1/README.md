# E-Learning Platform (Frontend)

A simple multi-page e-learning platform built with HTML, CSS, and JavaScript.

## Features

- **Dashboard**: Overview of user progress and recent quiz results
- **Courses**: Browse and mark courses as completed
- **Quiz**: Take JavaScript quizzes with grading and results
- **Profile**: View user statistics and completed courses
- **Data Persistence**: Uses localStorage to save progress

## Technologies

- HTML5
- CSS3 (with CSS Variables and Gradients)
- JavaScript (ES6+)

## How to Run

1. Clone or download the project files
2. Open `index.html` in a web browser
3. Navigate through the platform using the menu

## Project Structure

- `index.html` - Entry point (redirects to dashboard)
- `dashboard.html` - Main dashboard page
- `courses.html` - Courses listing and management
- `quiz.html` - Quiz interface
- `profile.html` - User profile page
- `style.css` - Main stylesheet
- `script.js` - Core application logic
- `courses.js` - Course data
- `quiz-data.js` - Quiz questions
- `storage.js` - Local storage management
- `tests/` - Unit tests for quiz logic
